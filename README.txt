# 🏋️ FitFusion Club – Business Website

A complete multi-page **fitness club business website** showcasing services, trainers, membership plans, programs, blogs, and more.  
This project demonstrates the ability to build a **professional, real-world business site** using **HTML5, CSS3, Bootstrap 5, JavaScript, and jQuery**.  
SEO meta tags are included in each page to make the website search engine–friendly and professional.

---

## 🚀 Features
- Responsive design with **Bootstrap 5**
- Multi-page structure (10+ pages)
- Hero section with CTA on Home Page
- Membership plans & pricing
- Trainer profiles with images
- Programs & schedules
- Blog section (10 posts)
- Gallery for events & activities
- Testimonials page with carousel
- Contact form
- FAQ section
- Privacy Policy & Terms pages
- Basic **SEO optimization** with meta tags

---

## 🛠️ Tech Stack
- **HTML5** – Semantic page structure  
- **CSS3 + Bootstrap 5** – Styling & responsive design  
- **JavaScript (jQuery)** – Interactivity  
- **Owl Carousel / Plugins** – For sliders & testimonials
